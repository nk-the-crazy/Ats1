2. Управление Категориями Задач

Доступ к Разделу имеют только пользователи с разрешением "Управление Тестовыми Задачами".
Основной функционал "категории задач", пруппировка тестовых вопросов.
1. Создание Категоии Задач
Пройдите по нижеуказанной ссылке (Меню --> Тестовые Задачи --> Добавить Категории):
Заполните все необходимые поля в красном.
а. Данные Категории
Укажите название ,описание
б. Подкатегории
Для добавления подкатегории виберите Категорию из списка и нажмите на "Добавить Категорию". Это функция добавит под-категорию на выбранный элемент.
в. Удаление Категории
Чтобы удалить категорию, нажмите "Удалить" . Категорию можно удалить только если он не содержит под-элементы и тестовые задачи.
