1. Управление Тестированием

Доступ к Разделу имеют только пользователи с разрешением "Управление-Тестированием".
При создании тестирования, система автоматически создает "Код входа". "Код входа" используется участниками для прохождения тестирования.
1. Создание Тестирования
Для создания Тестирования вам необходимо создать Задачи.
Пройдите по нижеуказанной ссылке (Меню --> Тестирование -> Регист. Тестирования.):
Заполните все необходимые поля в красном.
а. Основные Данные Тестирования
Укажите основные данные и период тестирования. После даты окончания тестирования автоматически блокируется (доступ закрыт).
б. Выбор участников
Выберите участников которым назначено тестирование. Только выбранные группы имеют доступ к тестированию. Тестирование можно создать для разных незавысимых групп.
в. Выбор тестовых задач
Добавьте Варианты вопросов и укажите уровень сложности и количество. Нижеуказанный пример выбирает 100 вопросов типа "Единичный Выбор" , (40 простых , 60 средней уровни сложности).
г. Выбор вопросов по категориям.
Выберите категории из списка или укажите "Выбрать Все". При выборке по категориям, система выбирает вопросы, по видам и количеству (пред. панель) именно по выбранным категориям.
