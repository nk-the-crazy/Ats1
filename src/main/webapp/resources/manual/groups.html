3. Управление Группами

Группы пользователей не имеют отношение (функционал) к безопасности системы. Они используются для контроля Участников Тестирования (подробная инфо. в разделе тестирование).
1. Создание Групп
Пройдите по нижеуказанной ссылке (Меню --> Настройки Системы --> Группы Польз.--> Добавить Группу):
Создайте группу указывая Имя и описание. Дальнейшие операции над группой можно произвести в разделе "Редактировать Группу".
----------------------------------------------------------
2. Редактировать Группу
Пройдите по нижеуказанной ссылке (Меню --> Настройки Системы --> Группы .--> Детали Группы --> Редактировать):
В разделе "Редактировать Группу" , нажмите на "Добавить Пользователя" чтобы выбрать из списка один или больше пользователей.
-------------------------------------------------------
