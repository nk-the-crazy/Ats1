Глава 3. Тестовые Задачи

Система поддерживает нижеуказанные виды Тестовых Задач (вопросов).
№ Вид Описание
1 Единичный Выбор Только один правильный ответ
2 Множественный Выбор Несколько правильных ответов
3 Короткий ответ Короткий ответ. Участник должен Написать (набрать) короткое слово
4 Эссе Участник должен написать эссе. Оценка таких видов задач не автоматизировано. Менеджер Тестирования должен оценить и прокоментировать ответ.
Тестовые задачи имеют нижеуказанные уровни сложности:
№ Вид Описание
1 Простой  
2 Средний  
3 Сложный
