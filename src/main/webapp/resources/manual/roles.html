1. Управление Ролями

1. Создание Ролей
Пройдите по нижеуказанной ссылке (Меню --> Настройки Системы --> Роли --> Добавить Роль):
В Разделе "Регистрация роли" добавьте из списка Разрешения которые бы хотите назначить для роли и выберите (укажите) детали разрешения на Чтение/Запись/Изменение/Удаление.
В нижеуказанном рисунке, добавлены два Разрешения только для чтения (пользователь с данным ролем , имеет права только для просмотра списка пользователей и групп)
----------------------------------------------------------
-------------------------------------------------------
Для участников Тестирования необходимо создать роль с разрешением только "Тестирование".
